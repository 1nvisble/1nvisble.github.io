标准数据类型

Python有五个标准的数据类型：

  • Numbers（数字）
  • String（字符串）
  • List（列表）
  • Tuple （元组）
  • Dictionary（字典）

数字

Python支持四种不同的数字类型，用于存储数值，是不可改变的数据类型，其意思是如果数字数据类型会分配一个新的对象（关于不可见数据类型，详情请见前面博客不可见数据类型文章）。

  • int（整型）
  • float（浮点型）
  • complex（复数）

不需要定义数字类型，其会自动识别

也可以使用del语句删除一些对象的引用。

字符串

其大致与C语言差不多，不过其字符列表有两种取值顺序。

  • 从左到右索引默认0开始的，最大范围是字符串长度少1
  • 从右到左索引默认-1开始的，最大范围是字符串开头
H e l l o
0 1 2 3 4

一些其他操作

  • 从字符在截取一段字符，采用[头下标:尾下标]来截取相应的字符串，其中下标是从 0 开始算起，可以是正数或负数，下标可以为空表示取到头或尾，此外它是“左闭右开”的形式，截取的子串包含头下标的字符，但不包含尾下标的字符；

  • *表示重复，+表示连接；

  • P字符串截取还可以有第三个参数，其表示的截取的步长，在索引 1 到索引 4 的区间内并设置为步长为 2（间隔1位）来截取字符串

元组

元组是另一个数据类型，类似于 List（列表）。

元组用 () 标识。内部元素用逗号隔开。但是元组不能二次赋值，赋完初值后，其就相当于只读列表。

字典

字典(dictionary)是除列表以外python之中最灵活的内置数据结构类型。列表是有序的对象集合，字典是无序的对象集合。

两者之间的区别在于：字典当中的元素是通过键来访问存取的，而不是通过下标访问存取，有点类似于java中的Map。

字典用”{ }”标识。字典由索引(key)和它对应的值value组成，用:来区分键和值，1 : 123中，1表示键，123为整型值。
